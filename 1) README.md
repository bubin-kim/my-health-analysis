# My-Health-Analysis
이 프로젝트는 저의 10개월간의 혈액검사를 기반으로 Free T4(유리싸이록신) 수치의 변화를 시각화하였습니다.
Pandas와 Matplotlib를 활용했습니다.  

### 상관관계 분석
약 복용량을 X에서 Y로 증량한 후, 약 N주 뒤부터 수치가 정상 범위 내로 안정화되는 것을 확인했습니다.
수치가 정상 범위(0.92~1.68)를 벗어났을 때 복용량을 조절한 결과, 목표 수치에 도달하는 데 걸린 시간은 약 M개월이었습니다.

### 변동성 확인
동일한 복용량을 유지하더라도 특정 시기에 수치가 급변한 지점을 발견했습니다. 이는 당시의 컨디션이나 다른 외부 요인이 영향을 줄 수 있음을 시사합니다.

### 시각화의 목적
자기 관리: 단순히 기록하는 것을 넘어, 내 몸이 특정 용량의 약에 어떻게 반응하는지 데이터로 파악하여 추후 진료 시 의료진과 더 정확하게 소통하기 위함.
데이터 활용: 산발적인 검사지 데이터를 시계열 데이터로 변환하여 장기적인 추세를 추적함.
